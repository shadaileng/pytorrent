# torrent

#### 介绍

python3 实现 torrent 下载

- 解析`torrent`文件
- 支持`http`下载`pieces`
- 支持多线程下载
- 支持断点续传

#### 安装教程

```bash
$ pip3 install git+https://github.com/shadaileng/pytorrent.git
```
